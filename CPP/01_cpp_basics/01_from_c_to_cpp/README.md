# 从 C 到 C++ 的过渡

本模块帮助您从C语言平滑过渡到C++，重点介绍C++相对于C的改进和新特性。

## 学习目标

- 理解C++与C的主要区别
- 掌握引用、const、auto等C++基础特性
- 学会使用C++的输入输出流
- 理解命名空间的作用

## 文件列表

| 文件 | 内容 | 推理加速场景 |
|------|------|--------------|
| `01_references.cpp` | 引用 vs 指针 | 高效参数传递 |
| `02_const_constexpr.cpp` | const/constexpr | 编译期常量 |
| `03_auto_decltype.cpp` | 类型推导 | 简化模板代码 |
| `04_namespaces.cpp` | 命名空间 | 模块化组织 |
| `05_overloading.cpp` | 函数重载 | 统一接口 |
| `06_new_delete.cpp` | 动态内存 | 内存管理基础 |
| `07_io_streams.cpp` | 输入输出流 | 调试与日志 |

## 编译方法

```bash
g++ -std=c++17 -o output filename.cpp
```

## 学习建议

1. 每个文件都可以独立编译运行
2. 仔细阅读注释，理解C++相对于C的改进
3. 尝试修改代码，观察行为变化
4. 完成每个文件末尾的练习题
