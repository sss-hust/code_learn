# 类与对象

本模块介绍C++面向对象编程的核心概念。

## 学习目标

- 理解类的定义和成员访问控制
- 掌握构造函数、析构函数
- 理解拷贝控制和运算符重载
- 学习继承与多态

## 文件列表

| 文件 | 内容 | 推理加速场景 |
|------|------|--------------|
| `01_class_basics.cpp` | 类定义、成员 | Tensor类封装 |
| `02_constructors.cpp` | 构造/析构函数 | 资源管理 |
| `03_copy_control.cpp` | 拷贝控制 | 避免意外拷贝 |
| `04_operator_overload.cpp` | 运算符重载 | 张量运算 |
| `05_inheritance.cpp` | 继承多态 | 算子基类设计 |
